Complaint Service Hub Prototype v0.3

Start:
1. Run run_from_python.bat for Python test, or build EXE with 01_BUILD_EXE_PYINSTALLER.bat.
2. Select company. InSightec password is 5963.
3. Select user. Shared users must enter actual personal user before Complaint input.

New in v0.3:
- Settings is visible for all users.
- UI language is selected automatically from company/country default.
- UI language can be changed any time from the top Language dropdown.
- Settings allows auto/manual language mode, default country, and default output mode.
- Update ZIP function confirmed and remains available.

Distribution:
Zip the whole complaint_service_hub folder and give it to users.
